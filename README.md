# Wireframe Prototype

Folder ini adalah prototype wireframe hitam-putih.

Cara pakai biasa:
1. Buka `index.html` langsung di browser.
2. Pilih Login ORMAWA/UKM atau Login Kompetisi Mandiri.
3. Registrasi memakai alur register + OTP.

Untuk HTML-to-Figma:
1. Upload folder ini ke GitHub Pages.
2. Buka `figma-export.html`.
3. Copy link screen yang dibutuhkan, misalnya `index.html?role=super_admin&page=dashboard`.
4. Paste link itu ke HTML-to-Figma. Parameter URL langsung membuka halaman setelah login.

File penting:
- `index.html` - prototype utama.
- `app.js` - role, navigasi, deep-link export, dan interaksi.
- `styles.css` - style wireframe.
- `wireframe-lock.css` - pengunci tampilan hitam-putih.
- `figma-export.html` - daftar link semua screen.
